# Breadcrumb

Breadcrumb is a navigation component that shows the current page's location within a navigational hierarchy.

See the [W3C documentation](https://www.w3.org/WAI/ARIA/apg/patterns/breadcrumb/examples/breadcrumb/) for information on accessibility.
