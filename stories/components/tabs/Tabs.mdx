# Tabs - Requires JS

<Description
  markdown={dedent`
    | Selector | Type | Description |
    | --- | --- | --- |
    | \`.tabs\` | Component | Container of tab-list and tab-content items |
    | \`.tabs-md\` | Modifier | Changes the size of the tab item to medium |
    | \`.tab-list\` | Sub Component | Container of tab items |
    | \`.tab\` | Sub Component | A tab item. Requires a \`data-target\` attribute that matches the id of the corresponding tab-content item |
    | \`.tab-active\` | Modifier | Set the active tab and displays the corresponding tab-content item |
    | \`.tab-content\` | Sub Component | A tab-content item. Requires an \`id\` that matches the data-target attribute of the corresponding tab item |
  `}
/>

<script src="../path/to/@bkbank/ui/dist/components/tabs.js"></script>
