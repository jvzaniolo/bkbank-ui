# Avatar

| Selector                | Type      | Description                        |
| ----------------------- | --------- | ---------------------------------- |
| `.avatar`               | Component | Base class                         |
| `.avatar-sm`            | Modifier  | Changes the size to small          |
| `.avatar-lg`            | Modifier  | Changes the size to large          |
| `.avatar-primary`       | Modifier  | Changes the color to primary       |
| `.avatar-active`        | Modifier  | Defines the active state           |
| `img`                   | Content   | Styles the image inside the avatar |
| `svg` \| `.avatar-icon` | Content   | Styles the icon inside the avatar  |

## Default

<div className="avatar mt-6">RM</div>

```html copy
<div class="avatar">RM</div>
```
