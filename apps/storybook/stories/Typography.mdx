# Typography

Here are all the typography tokens we extended from the Tailwind base theme.

To use them, combine with the proper Tailwind class.

For example, to change the font size to `fontSize.2xs`, apply the `.text-2xs` class.

## Font Family

- **fontFamily.montserrat:** `['Montserrat', 'sans-serif']`

For more information about font family, please refer to the Tailwind [Font Family](https://tailwindcss.com/docs/font-family) page.

## Font Sizes

- **fontSize.2xs:** `'0.625rem' // 10px`;

For more information about font sizes, please refer to the Tailwind [Font Size](https://tailwindcss.com/docs/font-size) page.
