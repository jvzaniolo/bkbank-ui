# Tokens

In this section, you can find all the tokens used in the design system that we extended from the Tailwind base theme.

To use them, combine with the proper Tailwind class.

For example, to change the background color to `colors.primary.pure`, apply the `.bg-primary-pure` class.

## Typography

For more information about font sizes, please refer to the Tailwind [typography](https://tailwindcss.com/docs/font-family) page.

### Font Sizes

- `.text-2xs`: 0.625rem - 10px
