# Tabs

A set of layered sections of content—known as tab panels—that are displayed one at a time.

See [Tabs - Radix UI](https://www.radix-ui.com/docs/primitives/components/tabs/) for the full documentation.

## Sizes

The size of the Tabs. Can be `md`, or `lg`. Default is `md`.
