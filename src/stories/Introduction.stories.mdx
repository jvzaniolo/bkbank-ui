# Welcome to our Design System

Some description to the Design System

## Installation

First, configure your project to use [Tailwind CSS](https://tailwindcss.com/) as a [PostCSS](https://postcss.org/) plugin.

Then, install the design system and the PostCSS plugin `postcss-import`:

```bash
npm install -D @bkbank/design-system postcss-import
```

## Usage

Follow these steps to use the design system in your project:

### Configuring Tailwind

In your `tailwind.config.js` file, import the design system tailwind preset.

See [Tailwind CSS Presets](https://tailwindcss.com/docs/presets) for more information.

```js
// ./tailwind.config.js
module.exports {
   presets: [require("@bkbank/design-system/tailwind-preset")],
  // ...
}
```

That will enable you to use classes like `bg-primary-pure` or `text-secondary-dark` in your project.

### Importing components

We use some PostCSS plugins to develop the components. In order to use them, you need to add those plugins to your PostCSS configuration.

Import the `postcss-import` and the `tailwindcss/nesting` plugins in your `postcss.config.js` file:

```js
// ./postcss.config.js
module.exports = {
  plugins: {
    "postcss-import": {},
    "tailwindcss/nesting": {},
    tailwindcss: {},
    autoprefixer: {},
  },
};
```

Then, change your `tailwind.css` file from this:

```css
/* ./tailwind.css */
@tailwind base;
@tailwind components;
@tailwind utilities;
```

To use the `@import` syntax and import the design system components.

See [Tailwind CSS build-time imports](https://tailwindcss.com/docs/using-with-preprocessors#build-time-imports) for more information.

```css
/* ./tailwind.css */
@import "tailwindcss/base";
@import "tailwindcss/components";
@import "@bkbank/design-system/components";
@import "tailwindcss/utilities";
```
