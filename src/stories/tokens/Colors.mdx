# Color Palettes

Here are all the colors we extended from the Tailwind base theme.

To use them, combine with the proper Tailwind class.

## Usage

Tailwind generates a set of utility classes for each color in your `theme.colors` configuration object. These classes can be used to apply color stylings to any element.

```html
<div class="bg-primary-pure text-high-light">...</div>
```

For more information about how to use them, please refer to the [Tailwind CSS Colors](https://tailwindcss.com/docs/customizing-colors) page.
